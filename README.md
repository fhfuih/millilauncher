# Millilauncher

A minimalist, line-oriented Minecraft launcher

# Requirements

Python 3.x (I' m using 3.5. But other versions should be fine.)

`pip install click`

# Usage

The most basic method:
`millilauncher launch 1.11.2`

More to be implemented.
